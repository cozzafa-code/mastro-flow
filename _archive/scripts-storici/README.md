# _archive/scripts-storici

Script one-shot storici (patch, fix, debug, setup) usati durante
lo sviluppo MASTRO da marzo a maggio 2026. Archiviati il 13/05/2026
per pulizia root prima del refactor dei 5 file mostro.

## Categorie

- **01-patch-componenti** — Patch one-shot a file TSX
- **02-fix-bug** — Fix specifici (encoding, posizioni, tap, etc.)
- **03-debug-scan** — Tool di ispezione (mojibake, hex, scan)
- **04-feature-add** — Script di aggiunta feature
- **05-split-storici** — Tentativi precedenti di split file mostro
- **06-setup-tool** — Setup one-shot (gmail, onboarding, gitignore)
- **07-audit** — Tool di analisi (hooks)

## Stato

Nessuno di questi script e' attivo. Non sono referenziati da
`app/`, `components/`, `lib/`, `hooks/` o `pages/`.
Tenuti per consultazione storica.
